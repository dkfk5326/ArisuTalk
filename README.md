## 🤝 협업 안내

이 저장소는 공개 오픈소스이며, 협업자는 다음 권한을 가집니다.

### ✅ 가능한 작업
- 브랜치 생성 및 수정
- Pull Request(PR) 작성 및 리뷰
- 승인 후 PR 머지
- 이슈 생성, 수정, 라벨 관리
- 코드 개선, 버그 수정, 문서화 작업

### 🚫 하면 안 되는 작업
- `main` 브랜치에 직접 푸시
- `main` 브랜치 삭제 또는 강제 푸시
- 저장소 설정 변경, 보호 규칙 해제
- 라이선스 조건에 어긋나는 코드 사용

### How TO Start Server

1. 프로젝트 디렉토리로 이동합니다.
   ```sh
   cd arisutalk
   ```
2. 서버를 시작합니다.
   ```sh
   bash start.sh
   ```
